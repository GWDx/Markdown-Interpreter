## Markdown-Interpreter

将 Markdown 转换为 HTML。



### 编译

> + g++, make
> + python 3, pyinstaller

#### 方案一

```bash
git clone https://github.com/GWDx/Markdown-Interpreter.git
cd Markdown-Interpreter
mkdir bin
make
```

#### 方案二

```bash
git clone https://github.com/GWDx/Markdown-Interpreter.git
cd Markdown-Interpreter
mkdir bin
cd src
make
```
